Contributing - COVID-19 India Data by DataMeet

Project is on Github.

We are open to contributions. Please read DataMeet Community Code of Conduct before you start and also read our license to familiarize yourself with CC4.0 under which all our and your contributions will be distributed.

Referencing the Source

For any change you make please link to a source. We prefer government websites or websites/apps like OpenStreeMap who share the data under ODBl. If they are generic then you can add it to the reference.md file.

How to contribute

  • Fork the project
  • Make changes
  • Send a pull request

Acceptance procedure

  • One of the core volunteer will cross-check/validate with the references you add to pull request
  • If everything is okay then will accept the pull request

Reporting error

If you have any issues with the maps, please report them on the github repository. Give details and reference. One of us will pick it up.

Anything else

Start an issue on the issue tracker or email on the list.
